# 课表生成器使用说明

## 功能介绍

本应用程序可以将Excel格式的课程表转换为两种格式：
1. **教室课表** - 适合教师和管理员查看的详细课表
2. **教室班牌** - 适合在教室门口显示的简洁课表

## 系统要求

- Python 3.7 或更高版本
- 所需依赖包：
  - Flask
  - pandas
  - openpyxl

## 安装步骤

1. 确保已安装Python 3.7或更高版本
2. 下载并解压本文件夹到任意位置
3. 打开命令提示符或PowerShell，进入解压后的文件夹
4. 安装依赖包：
   ```
   pip install flask pandas openpyxl
   ```

## 运行方法

双击运行 `run_app.bat` 文件，或在命令提示符中执行：

```
python combined_app.py
```

程序启动后，会在命令行中显示访问地址，通常为 `http://127.0.0.1:8088`

## 使用说明

1. 在浏览器中打开显示的访问地址
2. 选择要上传的Excel源文件
3. 选择转换类型（教室课表或教室班牌）
4. 输入要生成的周数
5. 点击"生成课表"按钮
6. 转换完成后，点击"下载课表"按钮下载生成的文件

## 注意事项

- 源文件必须是Excel格式（.xlsx或.xls）
- 源文件的格式应与系统要求的格式一致
- 生成的文件会保存在 `uploads` 文件夹中

## 故障排除

- 如果出现依赖包缺失错误，请运行 `pip install flask pandas openpyxl` 安装所需依赖
- 如果端口8088被占用，可以修改 `combined_app.py` 文件中的端口号
- 如果转换失败，请检查源文件格式是否正确
